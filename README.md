# 🧾 Laravel + Livewire 3 + Filament v4 Full-Stack POS System

A modern full-stack web application built with the latest technologies — **Laravel**, **Livewire 3 Starter Kit**, and **Filament v4** — demonstrating how to rapidly build interactive dashboards, CRUD operations, and a complete **POS (Point of Sale)** system.

---

## 🚀 Tech Stack

- ⚙️ **Laravel** – The PHP framework for modern web apps  
- ⚡ **Livewire 3** – Reactive full-stack components with no JavaScript required  
- 🧩 **Filament v4** – Elegant admin panels, forms, tables, and dashboards  
- 🎨 **TailwindCSS** – Utility-first CSS for modern design  
- 🔐 **Authentication** – Integrated via the Livewire Starter Kit  

---

## 🎯 What You’ll Learn

### 🧰 Setup & Authentication
- Start with a pre-configured **Livewire 3 Starter Kit**
- Implement and customize **user authentication**
- Configure middlewares and guards for secure access

### 🗂️ Filament Integration
Integrate and explore **Filament v4’s core packages** to handle admin and UI logic:
- **Filament Notifications**
- **Filament Schemas**
- **Filament Forms**
- **Filament Tables**
- **Filament Support**

### ✏️ CRUD Operations
Master essential CRUD actions with Filament & Livewire:
- Create new records with Filament Forms  
- Display data using Filament Tables  
- Update or delete records seamlessly  
- Handle validation, modals, and real-time updates  

### 🧾 POS Page Development
Build a fully functional **Point of Sale (POS)** interface with:
- 🔍 Item search  
- ➕ Add / ➖ Remove from cart  
- 🔄 Update cart quantities  
- 💰 Apply tax and discounts  
- 💵 Handle change calculation  

### 📊 Dashboard & Analytics
Use **Filament Widgets** to build dynamic dashboards:
- Visualize sales metrics  
- Track performance trends  
- Display live stats and KPIs
